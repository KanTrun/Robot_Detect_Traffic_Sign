# Firmware Sources

This directory contains tracked firmware logic extracted from working ESP builds.

- `esp32cam/test-esp32cam-phase05.ino`: ESP32-CAM sign sender firmware
- `esp32cam/model_data.h`: model header used by ESP32-CAM firmware
- `esp32s3/test-esp32s3-phase05-integration.ino`: ESP32-S3 integration firmware

Notes:
- Wi-Fi credentials are placeholders and must be set before flashing.
- Source was recovered from build cache and normalized for repository tracking.
